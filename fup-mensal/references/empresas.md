> Se uma URL redirecionar ou falhar, pedir ao usuário a nova URL e atualizar este arquivo.

## Estrutura no Drive

```
Grupo VR  (pasta-raiz)
  └── FUP Grupo VR — Mês/Ano  (pasta mensal — criar nova a cada período)
        ├── Carlos Bruno
        ├── Danilo Tomaz
        └── ...

Grupo Lazari  (pasta-raiz)
  └── FUP Grupo Lazari — Mês/Ano
        ├── Ian Turcato
        └── ...

Grupo Silva  (pasta-raiz)
  └── FUP Grupo Silva — Mês/Ano
        ├── Igor Occon
        └── ...
```

Para cada novo período: criar pasta mensal dentro da pasta-raiz do grupo → criar subpastas por closer → fazer upload dos xlsx.
